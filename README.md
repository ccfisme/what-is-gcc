## 在Linux和Mac分别实践  

### 普通用户  

* 一般默认是在用户目录～下，Mac在`/User/ccf`下；Linux在`/home/admin`下
* `cd /`后，Mac是在`/`下，Linux在`/`下
### root后用户
* 一般默认是在用户目录～下，Mac在`/User/ccf`下；Linux在`/root`下
* `cd /`后，Mac是在`/`下，Linux在`/`下
